# 常见题型和方法

极限：

- 指数复杂的式子，取对数
  - e.g. $\lim\limits_{n\to\infty}\sqrt[n]{n^a} = \lim\limits_{n\to\infty}e^{\frac{a}{n}\ln n} = 1$

和式极限：

- 求和（裂项、等比数列）
- 定积分近似：$\frac{1}{n} = \mathrm{d}x$
- 夹逼定理：放缩、定积分放缩

定积分：原函数、奇偶性、周期性、几何意义

中值定理：

- 构造辅助函数：一阶导乘原函数项、二阶导项、一阶项与原函数之和
- 出现两个中值变量，区间划分
- 看到 $f'(0) = 0$ 考虑泰勒展开

级数考题：

- 常数项级数的敛散性判断
- 收敛半径、收敛区间（开区间）、收敛域（需要讨论区间端点）的求解
  - 比值法、根值法
- 函数展开为幂级数
- 级数求和（重难点）
- 傅里叶级数的收敛定理（狄利克雷）
- 函数展开为傅里叶级数
